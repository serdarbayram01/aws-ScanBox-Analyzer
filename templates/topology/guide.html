Topology — Setup Guide
Required IAM permissions and configuration for network topology scanning

1 Overview

Topology scans your AWS account using read-only boto3 API calls to discover VPCs, subnets, gateways, peering connections, and all networking resources. It creates interactive architecture diagrams following AWS best practices visualization standards.

All checks are read-only. Topology never creates, modifies, or deletes any AWS resource.
